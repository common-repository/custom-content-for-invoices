WooCommerce Custom Content for Invoices plugin display custom contents or values into your invoices.

== Description ==

WooCommerce Custom Content for Invoices plugin display custom contents or values into your invoices. If you want edit your invoice and add some extra custom content in your invoice above invoice table or after invoice table. This plugin is help you.


= Features =

* Display custom contents Before invoice table
* Display custom contents After invoice table
* You can use HTML code in content box 


== Installation ==

1. Install automatically through the `Plugins`, `Add New` menu in WordPress, or upload the `woocommerce-custom-content-invoices` folder to the `/wp-content/plugins/` directory. 

2. Activate the plugin through the `Plugins` menu in WordPress. On the Admin screen use the Plugins side menu to select Customize Invoice under WooCommerce Menu to configure.  

3. Updates are automatic. Click on "Upgrade Automatically" if prompted from the admin menu. If you ever have to manually upgrade, simply deactivate, uninstall, and repeat the installation steps with the new version.
